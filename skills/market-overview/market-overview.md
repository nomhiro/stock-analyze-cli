# マーケット概況スキル

## トリガー例
- 「市場の概況を教えて」
- 「今日の相場は？」
- 「注目のイベント・ニュースは？」
- 「値上がり/値下がりランキングは？」

## 分析フレームワーク
主要指数、メジャー銘柄の値動き、直近の適時開示イベントをまとめて日次サマリーを提供する。

## 使用するMCPツール
- `data_major_stocks()` — メジャー銘柄リスト取得
- `market_quotes(symbols)` — 主要銘柄の株価一括取得
- `market_disclosures(limit=20)` — 直近の適時開示
- `viz_ranking_table(data, columns)` — ランキングテーブル生成

## ワークフロー
1. data_major_stocks でメジャー銘柄リスト取得
2. 日経225指数（^N225）、TOPIX（^TOPX）等の主要指数を市場ベンチマークとして market_quote で取得
3. market_quotes でメジャー銘柄の株価を一括取得
4. ランキング計算:
   - 値上がり率 TOP10
   - 値下がり率 TOP10
   - 出来高ランキング TOP10
5. market_disclosures で直近の重要開示（[revision], [earnings] 等）を取得
6. viz_ranking_table で値上がり/値下がり/出来高のHTMLテーブル生成
7. Markdownレポート作成

## 出力フォーマット
- **主要指数**: 日経225、TOPIX、マザーズ指数の水準と変動率
- **マーケット・サマリー**: 市場の方向感（1〜2文）
- **値上がり率ランキング** (Markdownテーブル + HTMLリンク)
- **値下がり率ランキング** (Markdownテーブル + HTMLリンク)
- **出来高ランキング** (Markdownテーブル + HTMLリンク)
- **注目の適時開示** (リスト、[revision]を優先)
- **本日のトピック・注意点** (あれば)

## 分析上の注意点
- 市場の方向感は定量的に記述（「下落基調」ではなく「日経平均 -0.8%、値下がり銘柄数 68%」等）
- 適時開示は [revision]（業績修正）を最優先
- 日付情報（market_quote の regularMarketTime）も明示
