# パイプライン2: 企業スクリーニング（Who Will Win）

構造変化テーマに対し、最も有利なポジションにある企業を特定する。経済的堀（Moat）の4軸評価、サプライチェーン位置特定、経営者ビジョン評価を組み合わせる。

## 基本原則

- **定量フィルタ → 定性分析の順序を厳守** — 財務ファクトで母集団を絞ってから、堀やビジョンを評価する
- **原文引用付き根拠** — 堀スコアの根拠は必ず一次ソース（10-K、IR、決算説明会）からの引用で示す
- **サプライチェーン位置を特定** — 「誰が最大の価値を取り込むか」を明確にする

## サプライチェーン位置

テーマ内で企業が占める位置を分類する。過去の事例では、インフラ層 / プラットフォーム層が最も大きな富を築いてきた。

| 位置 | 特徴 | 代表例 |
|------|------|--------|
| infrastructure | 必須インフラ（GPU、クラウド基盤等） | NVIDIA型 |
| platform | 開発者・ユーザーを束ねるプラットフォーム | AWS型 |
| end_product | 最終製品・サービス | Tesla型 |
| component_supplier | 高付加価値部材・装置 | 信越化学型 |

## 経済的堀（4軸、各0-5）

Warren Buffett の「moat（堀）」概念を定量化する。各軸は独立にスコアリングし、原文引用で根拠を示す。

1. **ネットワーク効果（networkEffect）**
   - ユーザーが増えるほど価値が増す構造か
   - 例: 決済ネットワーク、SNS、マーケットプレイス

2. **スイッチングコスト（switchingCost）**
   - 顧客が他社へ乗り換える障壁の高さ
   - 例: 業務システム、エコシステム依存、学習コスト

3. **規模の経済（scaleEconomies）**
   - 生産量増で単位コストが下がる構造、または広告・R&D が固定費として効く産業
   - 例: 半導体ファブ、流通、クラウドインフラ

4. **無形資産（intangibleAssets）**
   - ブランド、特許、規制ライセンス、データ資産
   - 例: 医薬品特許、高級ブランド、金融ライセンス

各スコアは0-5の6段階。0=不在、5=圧倒的。

## 経営者ビジョン評価

短期業績ではなく長期構造に投資できる経営者かを評価する。

- **長期 vs 短期のバランス** — 決算説明会での発言、中期経営計画の質
- **R&D 投資の方向性** — 売上高 R&D 比率、投資分野の一貫性
- **資本配分** — 自社株買い / 配当 / M&A / R&D のバランス、過去の配分実績

`longTermFocus` を 0-100 でスコアリング（100=長期志向、0=短期志向）。

## ワークフロー

1. **企業ロングリスト作成**
   - `WebSearch` でテーマ関連企業を10-20社リストアップ
   - クエリ例: 「AI半導体 主要企業 2026」「量子コンピュータ 上場企業」
   - 業界レポート、ETF 構成銘柄、アナリストレポートを参照

2. **銘柄コード解決**
   - 日本株: `market_search` で企業名→.Tサフィックス付きコードに変換
   - 米国株: ティッカーシンボルをそのまま使用

3. **定量フィルタ（財務データ取得）**
   - `market_fundamentals` で各社の財務データ取得
   - 一次スクリーニング条件:
     - 売上成長率（revenueGrowthYoY）: 直近3年平均で業界平均以上
     - 粗利率（grossMargin）: 業界平均以上（堀の代理指標）
     - 営業キャッシュフロー: プラス継続
   - この段階で半数以下に絞る

4. **堀4軸評価**
   - 残った企業について4軸を各0-5でスコアリング
   - 根拠は 10-K / 有価証券報告書 / IR資料からの原文引用

5. **サプライチェーン位置特定**
   - infrastructure / platform / end_product / component_supplier のいずれかを選択
   - テーマ内の他プレイヤーとの関係を簡潔に記述

6. **SEC 10-K 取得（米国企業）**
   - `market_sec_filing` で最新 10-K の Item 1（Business）、Item 1A（Risk Factors）を取得
   - 事業セグメント記述、競合優位性、リスク要因を抽出

7. **IR 資料取得**
   - `WebSearch` / `WebFetch` で決算説明会資料、統合報告書、中期経営計画を取得
   - 経営者の言葉を原文で記録

8. **経営者ビジョン評価**
   - `longTermFocus` を0-100でスコアリング
   - 根拠となる経営者発言・投資実績を記録

9. **総合スコアリングとランキング**
   - 総合スコア = テーマ関連度 × 堀平均 × 経営者長期志向 の正規化値
   - 5-10 社にランキング

```json
{
  "companies": [
    {
      "symbol": "銘柄コード（日本株は.Tサフィックス）",
      "name": "企業名",
      "market": "jp | us",
      "themeRelevance": 0-100,
      "supplyChainPosition": "infrastructure | platform | end_product | component_supplier",
      "moatScores": {
        "networkEffect": { "score": 0-5, "evidence": "原文引用" },
        "switchingCost": { "score": 0-5, "evidence": "..." },
        "scaleEconomies": { "score": 0-5, "evidence": "..." },
        "intangibleAssets": { "score": 0-5, "evidence": "..." }
      },
      "managementVision": { "longTermFocus": 0-100, "evidence": "..." },
      "financialHighlights": {
        "revenueGrowthYoY": null,
        "grossMargin": null,
        "operatingCFPositive": null,
        "per": null,
        "pbr": null
      },
      "overallScore": 0-100
    }

```

## 注意点

- 定量フィルタを通過しない企業は堀評価に進まない（順序厳守）
- evidence が「引用できない企業」は信頼性が低いため除外を検討
- 日本株は `.T` サフィックスを必ず付与（例: `7203.T`）
- 財務データが欠損している場合は null のまま残し、虚偽の数値を入れない
- themeRelevance が 50 未満の企業は出力から除外
